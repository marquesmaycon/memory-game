# Memory Game

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![JavaScript](https://img.shields.io/badge/JavaScript-Game-F7DF1E?style=for-the-badge&logo=javascript&logoColor=111)

Jogo da memoria com tela de login, cronometro e cartas tematicas de Rick and Morty. O projeto usa JavaScript puro para controlar embaralhamento, revelacao de cartas, validacao de pares e finalizacao da partida.

## Demo

[Acessar projeto online](https://manualdodev.github.io/memory-game/)

## Funcionalidades

- Tela inicial com nome do jogador.
- Cartas embaralhadas a cada partida.
- Comparacao de pares.
- Cronometro em tempo real.
- Mensagem final com tempo da partida.
- Layout separado para login e tabuleiro.

## Stack

| Tecnologia | Uso |
| --- | --- |
| HTML | Telas do jogo |
| CSS | Layout, cartas e animacoes |
| JavaScript | Logica do jogo |

## Como jogar localmente

Abra `index.html` no navegador, informe um nome e clique em `Play`.

## Estrutura

```text
.
|-- index.html
|-- pages/game.html
|-- css/
|-- js/
`-- images/
```

## Regras

1. Digite o nome do jogador.
2. Vire duas cartas por vez.
3. Ao acertar um par, as cartas permanecem reveladas.
4. O jogo termina quando todos os pares forem encontrados.
